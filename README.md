# WizzardCards

WizzardCards é um jogo de cartas desenvolvido em Python utilizando a biblioteca Pygame, como projeto da disciplina de Programação Aplicada.

O jogo possui sistema de combate baseado em cartas, com mecânica de ataque e defesa, além de condições de vitória e derrota.

---

# EXECUTAR O PROJETO (MODO DESENVOLVIMENTO)

## 1. Pré-requisitos
- Python 3 instalado
- pip instalado

---

## Criando e ativando ambiente virtual (venv)

### Linux / macOS
```bash
python -m venv venv
source venv/bin/activate
```

### Windows (CMD ou PowerShell)
```bash
python -m venv venv
venv\Scripts\activate
```

## Instalar dependências
```bash
pip install -r requirements.txt
```

## Executar o jogo
```bash
python main.py
```

## Observação
- Linux/macOS usa `source venv/bin/activate`
- Windows usa `venv\Scripts\activate`
- Não misture os comandos
