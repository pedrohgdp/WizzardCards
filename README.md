# WizzardCards

WizzardCards é um jogo de cartas desenvolvido em Python utilizando a biblioteca Pygame, como projeto da disciplina de Programação Aplicada.

O jogo possui sistema de combate baseado em cartas, com mecânica de ataque e defesa, além de condições de vitória e derrota.

---

# Instalação

## Linux

```bash
python -m venv venv
source venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Windows (CMD)

```cmd
python -m venv venv
venv\Scripts\activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Windows (PowerShell)

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
